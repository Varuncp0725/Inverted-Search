# Inverted Search

A C-based application that creates an inverted index for multiple text files and enables efficient word searching.

## Features

* Create an inverted index from multiple text files
* Search for a word across indexed files
* Display the list of files containing the searched word
* Save the index to a database file
* Load an existing database for searching
* Implemented using hash tables and linked lists

## Technologies Used

* C Programming
* Hash Tables
* Linked Lists
* File Handling
* Data Structures
